    Hello world #

    First Project #

    Let’s create the first project.

    Create a folder to use as project folder.

    mkdir helloworld; cd helloworld

    Then create your first file main.go and type the following code:

    package main

import "fmt"

func main() {
	fmt.Println("Hello world")
}

    To run it, just type:

    go run main.go

    So let’s understand the code.

    package #

    The first line of code is the package name. The main package is the one that will contain the main method. Only this package is allowed to start a go application.

    import #

    The next lines we have the imports. We can refer a list of dependencies, just creating a list of imports inside parentheses:

    import (
    "fmt"
    "math"
    "strconv"
)

    The list should be one in each line and typed between quotation marks "".

    As expected, we can’t have imports with same name, so we must use an alias for some import.

    import (
    "fmt"
    myfmt "my-custom-lib/fmt"
)

    By convencion, the import names should be all in small letter not camel case

    func main #

    To start a go app, we need a main function, that follows the contract.

    func main() {
	...
}

    Named “main”, no args and no returns.
